Welcome to the PropWise CRM developer documentation. This site covers the full platform architecture, module specifications, access control system, and frontend patterns.

## Architecture overview

PropWise CRM is a multi-tenant real estate CRM platform built with:

- **Backend:** NestJS 10 + TypeScript + MikroORM + PostgreSQL (Neon)
- **Frontend:** Next.js 15 (App Router) + TanStack Query + Jotai + Tailwind CSS
- **Real-time:** Socket.io for notifications and messaging
- **Auth:** JWT + Passport with per-organization sessions
- **Storage:** Cloudflare R2 for file uploads
