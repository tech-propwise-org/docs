## Overview

This document provides a **complete, code-verified implementation plan** for rolling out the Free/Pro/Business subscription tiers with:

- **Evergreen 90-day trials** (card-upfront) on Pro & Business
- **Unified AI credit wallet** with per-user ceilings and personal credit support
- **Storage metering** (5GB/50GB/100GB base + 25GB purchase packs)
- **Entity caps** (10/200/unlimited) for Free tier
- **Consolidated seat model** (single per-agent SKU, retiring manager/agent split)
- **OpenAI-only billable traffic** routed through per-org project keys

The strategy identified **12 gaps**; code-level exploration confirmed all of them. This plan groups them into **11 phases** (Phase 0–10), backend-first, then frontend.

- Zero out `managerSeats*` (no DDL, just data migration)
- Avoids new columns

</Tab>

<Tab title="Credit Metering">

### Unified Wallet Owner
**Decision**: Subscription module owns wallet via new `CreditMeteringService` facade.

**Facade wraps**:
- `CreditService`
- Consolidated per-user allocation table
- `CreditPurchase`

### Ledger Consolidation
Two AI budget planes (`AiUsageAccount`, `AiCreditUsage`) consolidated into **ONE store** behind facade.

**Dependency direction**:
- OpenAI project provisioning/key resolution stays in AI module
- Surfaces depend on subscription for credits (correct direction)
- Pass plain `SpendContext` so subscription never imports RBAC/messaging

### Budget Model
**Shared org pool** with per-user **CEILINGS**, NOT reserved carve-outs.

**User ceiling caps** how much of shared pool that user may personally consume:
```typescript
effectiveUserOrgDraw = min(
  userLimit − userOrgSpent,
  orgPoolRemaining
)
```

**Ceilings MAY oversubscribe** the allowance:
- Remove agent's `allowance − Σ budgets` math
- Remove `Σ caps ≤ allowance` validation in `upsertUserBudget`

### Personal Credits
**Users may buy personal credits** (separate from org-set limit).

**Consumption order**: org-first, then personal.

**Routing**:
- Owner/Admin + org-level-channel AI → org pool
- Regular agent (assistant/Propilot, personal-channel AI) → user's allocation
- Personal credits = after-org fallback for ANY human actor (owner/admin included)
- Only org-level-channel auto-reply has no human actor (no personal layer)

</Tab>

<Tab title="Model Providers & Costs">

### Billable Traffic
**OpenAI-only**, routed through per-org OpenAI project key (agent's existing pattern, extended to assistant).

**Non-billable**:
- Anthropic: stays for eval judge only
- Gemini: **dropped**
- Embeddings: already OpenAI

**Telemetry**: OpenAI per-project usage is true-$ telemetry; customers billed in credits via cost map (not raw USD).

### Messaging Credits
**Decision**: Collapse `CreditType.MESSAGING` into single `CreditType.AI` wallet.

Messaging auto-reply draws from AI balance.

### Unit Valuation
**Charge**: 300 credits per valuation call (sum when turn triggers multiple).

**Prompt charge waiver**: When a turn performs a valuation, that turn's Propilot/prompt charge is **waived**.

<Note>
Waiver is assistant-orchestrator-specific (N/A on agent path unless agent gains valuation tool).
</Note>

</Tab>

<Step title="Guard Logic">

**Before create**:
```typescript
async assertCanCreate(
  orgId: string,
  entity: 'lead' | 'contact' | 'company',
  source: 'user' | 'inbound' | 'import',
) {
  if (!this.isSubscriptionEnforcementEnabled()) return;
  
  const plan = await this.getPlan(orgId);
  const limit = plan.limits[`max${capitalize(entity)}s`];
  
  if (limit === null) return; // Unlimited
  
  const count = await this.countNonImported(orgId, entity);
  
  if (count >= limit) {
    if (source === 'user') {
      // Hard-block user-initiated creates
      throw new ForbiddenException(
        `Free plan limited to ${limit} ${entity}s. Upgrade to add more.`,
      );
    } else if (source === 'inbound') {
      // Soft-degrade: create + emit signal
      await this.eventEmitter.emit('entity.cap.exceeded', {
        orgId,
        entity,
        limit,
      });
      // Allow create to proceed
    }
    // Import source: skip check entirely
  }
}
```

<Step title="5.5: Spend Logic (Org-First, Personal Fallback)">

**Implementation**:
```typescript
async spend(
  orgId: string,
  userId: string | null,
  amount: number,
  context: SpendContext,
) {
  if (!this.isSubscriptionEnforcementEnabled()) {
    // When enforcement OFF, log but don't deduct
    await this.logSpend(orgId, userId, amount, context);
    return;
  }
  
  if (!userId) {
    // Org-level spend (e.g., auto-reply) — org pool only
    await this.deductOrgPool(orgId, amount, context);
    return;
  }
  
  // User-initiated spend: org-first, then personal
  const allocation = await this.getAllocation(orgId, userId);
  const orgAvailable = await this.getOrgAvailableForUser(orgId, userId, allocation);
  
  if (orgAvailable >= amount) {
    // Deduct from org pool
    await this.deductOrgPool(orgId, amount, context);
    await this.incrementUserOrgSpent(allocation, amount);
  } else {
    // Deduct orgAvailable from org, rest from personal
    const personalNeeded = amount - orgAvailable;
    
    if (allocation.personalCredits < personalNeeded) {
      throw new InsufficientCreditsError(
        `Need ${personalNeeded} personal credits, have ${allocation.personalCredits}`,
      );
    }
    
    if (orgAvailable > 0) {
      await this.deductOrgPool(orgId, orgAvailable, context);
      await this.incrementUserOrgSpent(allocation, orgAvailable);
    }
    
    allocation.personalCredits -= personalNeeded;
    allocation.personalSpent += personalNeeded;
    await this.userAllocationRepo.flush();
  }
}

private async getOrgAvailableForUser(
  orgId: string,
  userId: string,
  allocation: UserCreditAllocation,
): Promise<number> {
  const orgPool = await this.getOrgBalance(orgId);
  
  if (allocation.orgCeiling === null) {
    // No ceiling = full pool access
    return orgPool;
  }
  
  // Ceiling caps this user's draw
  const userRemaining = allocation.orgCeiling - allocation.orgSpent;
  return Math.min(userRemaining, orgPool);
}
```

  source: 'user' | 'inbound',
