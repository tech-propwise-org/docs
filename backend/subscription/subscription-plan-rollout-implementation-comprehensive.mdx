# Subscription Plan Rollout - Implementation Guide

This guide details the complete implementation plan for rolling out the Free/Pro/Business subscription packaging strategy with phased backend and frontend changes. It addresses 12 identified gaps and implements evergreen trials, unified credit metering, entity caps, and seat management.

<Note>
This implementation plan is based on `SUBSCRIPTION_PACKAGING_STRATEGY.md` and incorporates two code-level verification passes that confirmed current-state facts and refined the execution plan.
</Note>

### Key Decisions

<AccordionGroup>
  <Accordion title="Internal vs Display Tier Names">
    Keep the internal tier ID as `PROFESSIONAL` but display it as "Pro" throughout the UI. This avoids renaming the enum across DB rows, Stripe metadata, DTOs, and frontend types. The `STARTER` tier is retired completely.
  </Accordion>

  <Accordion title="Seat Model Consolidation">
    Collapse the manager/agent two-SKU seat model into a single per-agent seat SKU per tier. Repurpose existing `agentSeats*` columns as the unified seat counter and zero out `managerSeats*` (no DDL changes needed).
  </Accordion>

  <Accordion title="Credit Wallet Ownership">
    The unified wallet is owned by the subscription module via a new `CreditMeteringService` facade. This consolidates the two AI budget planes (`AiUsageAccount`, `AiCreditUsage`) into ONE store, eliminating independent cost planes.
  </Accordion>

  <Accordion title="Budget Model - Shared Pool with Ceilings">
    Implement a shared org pool with per-user CEILINGS (not reserved carve-outs). A user ceiling caps how much of the shared pool that user may personally consume:
    
    ```
    effective_user_draw = min(userLimit - userOrgSpent, orgPoolRemaining)
    ```
    
    Ceilings MAY oversubscribe the allowance—remove the Σ budgets and Σ caps ≤ allowance validation.
  </Accordion>

  <Accordion title="Personal Credits Fallback">
    Users may buy personal credits separate from org-set limits. Consumption order:
    1. Org pool first
    2. Personal credits as fallback
    
    Owner/Admin and org-level-channel AI route to org pool. Regular agent assistant/Propilot and personal-channel AI route to user allocation. Personal credits work as after-org fallback for ANY human actor (including owner/admin).
  </Accordion>

  <Accordion title="AI Provider Strategy">
    - **OpenAI**: Billable model traffic only, routed through per-org OpenAI project key
    - **Anthropic**: Eval judge only
    - **Gemini**: Dropped
    - **Embeddings**: OpenAI
    
    OpenAI's per-project usage provides true-$ telemetry; customers are billed in credits via cost map (not raw USD).
  </Accordion>

  <Accordion title="Entity Cap Source-of-Record">
    - **Lead**: Reuse `Lead.leadSource === LeadSource.IMPORTED` (no new column)
    - **Contact/Deal/Company**: New `isImported` boolean
    
    Imported rows are excluded from cap counts. Auto-created companies during contact import MUST be stamped `isImported = true`.
  </Accordion>

  <Accordion title="Entity Cap Count Predicate">
    Count predicate: `isImported = false AND isDeleted = false`
    
    **Archived rows STILL count toward the cap**—archiving never frees a slot (only delete does). Keep `isDeleted` filter ON; disable `active` filter (`{ isArchived: false }`) on count queries.
  </Accordion>

  <Accordion title="Cap Enforcement Strategy">
    - **User-initiated creates**: Hard-block with 403 (manual, create-from-listing, manual API)
    - **Inbound WhatsApp/Meta**: Soft-degrade (create lead + emit over-limit signal, never drop live demand)
    - **Bulk import**: Skip cap enforcement entirely
    - **Storage uploads**: Hard-block user uploads; soft-degrade inbound messaging media
    
    Accept bounded check-then-create overshoot (no row locks), consistent with credit soft-guard.
  </Accordion>

  <Accordion title="Permission Model">
    - **Org buying + subscription management**: OWNER-only (`system.owner`)
    - **Per-user ceiling distribution**: Admin-level
    - **Route**: Single `/settings/billing` route (OWNER-gated via `system.owner`, NOT admin)
  </Accordion>
</AccordionGroup>

### 1.3 One-Time Purchase Handler

```typescript
// stripe-webhook.service.ts (NEW)
async handleChargeSucceeded(event: Stripe.Event): Promise<void> {
  const charge = event.data.object as Stripe.Charge;
  
  // Only process one-time purchases (not subscription invoices)
  if (charge.invoice) {
    return;  // Subscription charge, handled via invoice.payment_succeeded
  }

  const session = await this.stripe.checkout.sessions.retrieve(
    charge.metadata.session_id,
    { expand: ['line_items'] }
  );

  if (session.mode !== 'payment') {
    return;  // Not a one-time purchase
  }

  const orgId = session.metadata.organization_id;
  const userId = session.metadata.user_id;
  const productType = session.metadata.product_type;

  switch (productType) {
    case 'credit_pack':
      await this.fulfillCreditPack(orgId, userId, session);
      break;
    case 'storage_pack':
      await this.fulfillStoragePack(orgId, session);
      break;
    default:
      this.logger.warn(`Unknown product type: ${productType}`);
  }
}

private async fulfillCreditPack(
  orgId: string,
  userId: string,
  session: Stripe.Checkout.Session
): Promise<void> {
  const lineItem = session.line_items.data[0];
  const credits = parseInt(lineItem.price.metadata.credits, 10);

  // Record the purchase
  await this.creditPurchaseRepository.create({
    organizationId: orgId,
    userId,
    stripeChargeId: session.payment_intent as string,
    stripeSessionId: session.id,
    credits,
    amountPaid: session.amount_total / 100,  // Convert cents to AED
    currency: session.currency.toUpperCase(),
    purchasedAt: new Date(),
  });

  // Credit the user's PERSONAL wallet (not org pool)
  await this.creditMeteringService.addPersonalCredits(userId, credits, {
    source: 'purchase',
    referenceId: session.id,
  });

  this.logger.log(
    `Fulfilled ${credits} credit pack purchase for user ${userId} in org ${orgId}`
  );
}

private async fulfillStoragePack(
  orgId: string,
  session: Stripe.Checkout.Session
): Promise<void> {
  const lineItem = session.line_items.data[0];
  const storageGB = parseInt(lineItem.price.metadata.storage_gb, 10);

  await this.subscriptionUsageService.addStorageAllowance(orgId, storageGB);

  this.logger.log(
    `Added ${storageGB}GB storage pack to org ${orgId}`
  );
}
```

```typescript
// subscription-activation.service.ts (NEW/REFACTOR)
private buildActivationParamsFromCheckoutMetadata(
  stripeSubscription: Stripe.Subscription,
  checkoutMetadata: Record<string, string>
): SubscriptionActivationParams {
  const tier = checkoutMetadata.tier as PlanTier;
  const planDef = PLAN_DEFINITIONS[tier];

  // Derive trial state from Stripe subscription
  const isTrialing = stripeSubscription.status === 'trialing';
  const trialEndsAt = isTrialing && stripeSubscription.trial_end
    ? new Date(stripeSubscription.trial_end * 1000)
    : null;

  return {
    organizationId: checkoutMetadata.organization_id,
    tier,
    stripeSubscriptionId: stripeSubscription.id,
    stripeCustomerId: stripeSubscription.customer as string,
    status: this.mapStripeStatus(stripeSubscription.status),
    isTrialing,
    trialEndsAt,
    currentPeriodStart: new Date(stripeSubscription.current_period_start * 1000),
    currentPeriodEnd: new Date(stripeSubscription.current_period_end * 1000),
    agentSeatsTotal: this.extractSeatCount(stripeSubscription),
    managerSeatsTotal: 0,  // Deprecated, always 0
  };
}

private mapStripeStatus(stripeStatus: Stripe.Subscription.Status): SubscriptionStatus {
  const statusMap: Record<string, SubscriptionStatus> = {
    'trialing': SubscriptionStatus.TRIALING,
    'active': SubscriptionStatus.ACTIVE,
    'past_due': SubscriptionStatus.PAST_DUE,
    'canceled': SubscriptionStatus.CANCELED,
    'unpaid': SubscriptionStatus.UNPAID,
    'incomplete': SubscriptionStatus.INCOMPLETE,
    'incomplete_expired': SubscriptionStatus.INCOMPLETE_EXPIRED,
  };
  return statusMap[stripeStatus] || SubscriptionStatus.ACTIVE;
}
```

### 2.3 Activation Service

```typescript
// subscription-activation.service.ts
async activateSubscription(params: SubscriptionActivationParams): Promise<void> {
  const org = await this.em.findOneOrFail(Organization, params.organizationId);

  // Create or update subscription record
  let subscription = await this.em.findOne(OrganizationSubscription, {
    organization: org,
  });

  if (!subscription) {
    subscription = this.em.create(OrganizationSubscription, {
      organization: org,
      tier: params.tier,
      stripeSubscriptionId: params.stripeSubscriptionId,
      stripeCustomerId: params.stripeCustomerId,
      status: params.status,
      isTrialing: params.isTrialing,
      trialEndsAt: params.trialEndsAt,
      currentPeriodStart: params.currentPeriodStart,
      currentPeriodEnd: params.currentPeriodEnd,
    });
  } else {
    subscription.tier = params.tier;
    subscription.status = params.status;
    subscription.isTrialing = params.isTrialing;
    subscription.trialEndsAt = params.trialEndsAt;
    subscription.currentPeriodStart = params.currentPeriodStart;
    subscription.currentPeriodEnd = params.currentPeriodEnd;
  }

  await this.em.persistAndFlush(subscription);

  // Initialize usage record
  await this.subscriptionUsageService.initializeUsage(
    params.organizationId,
    params.agentSeatsTotal,
    params.tier
  );

  this.logger.log(
    `Activated ${params.tier} subscription for org ${params.organizationId}` +
    (params.isTrialing ? ` (trial ends ${params.trialEndsAt})` : '')
  );
}
```

```typescript
// entity-cap.service.ts
import { Injectable } from '@nestjs/common';
import { EntityManager } from '@mikro-orm/core';

export enum EntityType {
  COMPANY = 'company',
  DEAL = 'deal',
  CONTACT = 'contact',
  INVENTORY_UNIT = 'inventory_unit',
}

@Injectable()
export class EntityCapService {
  constructor(private readonly em: EntityManager) {}

  async assertCanCreate(
    orgId: string,
    entityType: EntityType,
    skipForImport: boolean = false
  ): Promise<void> {
    if (skipForImport) {
      return;  // Bulk import bypasses caps
    }

    const subscription = await this.getSubscription(orgId);
    const limit = this.getLimit(subscription.tier, entityType);

    if (limit === null) {
      return;  // Unlimited for Pro/Business
    }

    const currentCount = await this.countEntities(orgId, entityType);

    if (currentCount >= limit) {
      throw new EntityCapExceededError(entityType, limit, currentCount);
    }
  }

  private async countEntities(
    orgId: string,
    entityType: EntityType
  ): Promise<number> {
    // Count predicate: isImported = false AND isDeleted = false
    // Archived rows STILL count (disable active filter)
    const baseWhere = {
      organization: orgId,
      isDeleted: false,
    };

    switch (entityType) {
      case EntityType.COMPANY:
        return this.em.count(Company, {
          ...baseWhere,
          isImported: false,
        });

      case EntityType.DEAL:
        return this.em.count(Deal, {
          ...baseWhere,
          isImported: false,
        });

      case EntityType.CONTACT:
        return this.em.count(Contact, {
          ...baseWhere,
          isImported: false,
        });

      case EntityType.INVENTORY_UNIT:
        return this.em.count(InventoryUnit, {
          ...baseWhere,
          // No isImported column on inventory
        });

      default:
        throw new Error(`Unknown entity type: ${entityType}`);
    }
  }

  private getLimit(tier: PlanTier, entityType: EntityType): number | null {
    const planDef = PLAN_DEFINITIONS[tier];

    switch (entityType) {
      case EntityType.COMPANY:
        return planDef.limits.maxCompanies;
      case EntityType.INVENTORY_UNIT:
        return planDef.limits.maxInventoryUnits;
      case EntityType.DEAL:
      case EntityType.CONTACT:
        // Future: add to plan limits
        return null;
      default:
        return null;
    }
  }

  private async getSubscription(orgId: string): Promise<OrganizationSubscription> {
    const sub = await this.em.findOne(OrganizationSubscription, {
      organization: orgId,
    });

    if (!sub) {
      // Default to Free for orgs without explicit subscription
      return { tier: PlanTier.FREE } as OrganizationSubscription;
    }

    return sub;
  }

  async getCapStatus(
    orgId: string,
    entityType: EntityType
  ): Promise<CapStatus> {
    const subscription = await this.getSubscription(orgId);
    const limit = this.getLimit(subscription.tier, entityType);
    const current = await this.countEntities(orgId, entityType);

    return {
      entityType,
      limit,
      current,
      remaining: limit === null ? null : Math.max(0, limit - current),
      isUnlimited: limit === null,
      isAtLimit: limit !== null && current >= limit,
    };
  }
}

// entity-cap-exceeded.error.ts
export class EntityCapExceededError extends Error {
  constructor(
    public readonly entityType: EntityType,
    public readonly limit: number,
    public readonly current: number
  ) {
    super(
      `Entity cap exceeded: ${entityType} (limit: ${limit}, current: ${current})`
    );
    this.name = 'EntityCapExceededError';
  }
}
```

```typescript
// credit-metering.service.ts
@Injectable()
export class CreditMeteringService {
  constructor(
    private readonly em: EntityManager,
    private readonly allocationRepo: EntityRepository<UserCreditAllocation>,
    private readonly subscriptionService: SubscriptionService,
    private readonly configService: ConfigService,
  ) {}

  /**
   * Get effective credit balance for a spend context
   * Respects isSubscriptionEnforcementEnabled() flag
   */
  async getBalance(context: SpendContext): Promise<CreditBalance> {
    if (!this.isEnforcementEnabled()) {
      return this.getUnlimitedBalance();
    }

    const [orgBalance, userAllocation] = await Promise.all([
      this.getOrgPoolBalance(context.organizationId),
      this.getUserAllocation(context.userId, context.organizationId),
    ]);

    // Org-level channel or no user context → org pool only
    if (!context.userId || context.channel === 'org_level') {
      return {
        source: 'org_pool',
        available: Math.max(0, orgBalance),
        orgPool: orgBalance,
        personalCredits: 0,
      };
    }

    // Calculate user's effective draw from org pool
    const userOrgDraw = this.calculateUserOrgDraw(
      userAllocation,
      orgBalance
    );

    return {
      source: 'user_allocated',
      available: userOrgDraw + userAllocation.personalCreditsBalance,
      orgPool: userOrgDraw,
      personalCredits: userAllocation.personalCreditsBalance,
    };
  }

  /**
   * Deduct credits from the appropriate balance(s)
   * Order: org pool first, then personal credits
   */
  async deductCredits(
    context: SpendContext,
    amount: number,
    metadata: DeductionMetadata
  ): Promise<DeductionResult> {
    if (!this.isEnforcementEnabled()) {
      return { deducted: amount, source: 'unlimited' };
    }

    const balance = await this.getBalance(context);

    if (balance.available < amount) {
      throw new InsufficientCreditsError(amount, balance.available);
    }

    // Deduct from org pool first
    let remaining = amount;
    const deductions: DeductionRecord[] = [];

    if (balance.orgPool > 0 && remaining > 0) {
      const orgDeduct = Math.min(remaining, balance.orgPool);
      await this.deductFromOrgPool(
        context.organizationId,
        context.userId,
        orgDeduct,
        metadata
      );
      deductions.push({ source: 'org_pool', amount: orgDeduct });
      remaining -= orgDeduct;
    }

    // Deduct from personal credits if needed
    if (balance.personalCredits > 0 && remaining > 0) {
      const personalDeduct = Math.min(remaining, balance.personalCredits);
      await this.deductFromPersonalCredits(
        context.userId,
        personalDeduct,
        metadata
      );
      deductions.push({ source: 'personal', amount: personalDeduct });
      remaining -= personalDeduct;
    }

    return {
      deducted: amount,
      source: deductions.length === 1 ? deductions[0].source : 'mixed',
      breakdown: deductions,
    };
  }

  private calculateUserOrgDraw(
    allocation: UserCreditAllocation,
    orgBalance: number
  ): number {
    if (!allocation.orgCreditCeiling) {
      // No ceiling = can use full org balance
      return orgBalance;
    }

    // User draw = min(ceiling - consumed, org remaining)
    const userRemaining = Math.max(
      0,
      allocation.orgCreditCeiling - allocation.orgCreditsConsumed
    );

    return Math.min(userRemaining, orgBalance);
  }

  private isEnforcementEnabled(): boolean {
    return this.configService.get('subscription.enforcementEnabled', true);
  }

  private getUnlimitedBalance(): CreditBalance {
    // When enforcement is OFF, return sentinel value
    return {
      source: 'unlimited',
      available: Number.MAX_SAFE_INTEGER,
      orgPool: Number.MAX_SAFE_INTEGER,
      personalCredits: 0,
    };
  }
}
```

```typescript
private async getOrgPoolBalance(orgId: string): Promise<number> {
  const usage = await this.em.findOneOrFail(SubscriptionUsage, {
    organization: orgId,
  });

  const subscription = await this.em.findOne(OrganizationSubscription, {
    organization: orgId,
  });

  const planDef = PLAN_DEFINITIONS[subscription?.tier || PlanTier.FREE];

  // Monthly allowance = aiCreditsPerMonth × seat count
  // (Requires Phase 9's getSeatCount to be implemented first)
  const seatCount = await this.subscriptionService.getSeatCount(orgId);
  const monthlyAllowance = planDef.aiCreditsPerMonth * seatCount;

  // Purchased credits (one-time packs)
  const purchasedCredits = await this.getPurchasedOrgCredits(orgId);

  // Total pool = allowance + purchased - consumed
  const totalPool = monthlyAllowance + purchasedCredits - usage.aiCreditsUsed;

  return Math.max(0, totalPool);
}

private async getPurchasedOrgCredits(orgId: string): Promise<number> {
  // Aggregate one-time credit pack purchases for this org
  const purchases = await this.em.find(CreditPurchase, {
    organization: orgId,
  });

  return purchases.reduce((sum, p) => sum + p.credits, 0);
```
