## Overview

This rollout introduces a three-tier subscription model (Free/Pro/Business) with AED pricing, 90-day evergreen trials, and unified credit metering. The implementation consolidates legacy budget systems, introduces entity caps for the Free tier, and implements per-seat credit allowances.

### Core Decisions

<AccordionGroup>
<Accordion title="Tier Naming & Internal IDs">
- **Internal ID**: Keep `professional` in database/enums
- **Display Name**: "Pro" in UI
- **Rationale**: Avoids renaming enum across DB rows, Stripe metadata, DTOs, and frontend types
- **Starter Retirement**: Hard remove via data migration (repoint to `professional`, then delete enum)
</Accordion>

<Accordion title="Seat Model Consolidation">
- **Change**: Collapse manager/agent two-SKU model into single per-agent seat SKU per tier
- **Implementation**: Repurpose existing `agentSeats*` columns as single seat counter
- **Manager Seats**: Zero out `managerSeats*` columns (no DDL)
- **Backfill Required**: Existing rows need `agentSeatsTotal = manager + agent`
</Accordion>

<Accordion title="Unified Credit Wallet Architecture">
- **Owner**: Subscription module via new `CreditMeteringService` facade
- **Consolidation**: Merge `AiUsageAccount` + `AiCreditUsage` into single store
- **Dependency Direction**: AI surfaces depend on subscription for credits (correct module layering)
- **Budget Model**: Shared org pool with per-user ceilings (NOT reserved carve-outs)
- **Consumption Order**: Org-first, then personal fallback
</Accordion>

<Accordion title="Entity Cap Source-of-Record">
- **Lead**: Reuse `Lead.leadSource === LeadSource.IMPORTED` (no new column)
- **Contact/Deal/Company**: New `isImported` boolean column
- **Auto-Created Companies**: Stamped `isImported = true` when contact importer creates by name
- **Count Predicate**: Non-imported AND `isDeleted = false` (archived rows still count)
- **Enforcement**: Hard-block user creates (403), soft-degrade WhatsApp/Meta auto-create, skip bulk import
</Accordion>

<Accordion title="AI Model & Billing Routing">
- **Billable Traffic**: OpenAI-only via per-org project key
- **Anthropic**: Retained for eval judge only
- **Gemini**: Dropped
- **Embeddings**: OpenAI
- **Telemetry**: OpenAI per-project usage as true-$ data
- **Customer Billing**: Credits via cost map (not raw USD)
</Accordion>
</AccordionGroup>

## Code Verification Results

<Info>
Two verification passes confirmed current-state facts and refined the implementation plan. Key findings folded into phases below.
</Info>

### Pass 1 Corrections

- **AI Budget Gate**: LIVE today (blocks calls), not telemetry-only—Phase 5 must treat as replacement
- **USD Ledger**: Lives in `AiCreditUsage`/`AiUsageAccount`, not `aiCreditsUsed` (numeric field has no writer)
- **Gate Markers**: Four `SUBSCRIPTION_GATE` blocks in `ai-agent-runtime.service.ts`, not three
- **Webhook Routing**: One-time credit-pack purchases need net-new webhook path
- **Seat Count**: `countSeatsByType` already uses cross-module raw SQL—consolidate to `getSeatCount`
- **Trial UX**: `trialEndsAt` DTO field + static badge + pack display exist; countdown/banner/top-up are new
- **Free Tier Credits**: 20 in current code vs 2000 proposed—intentional ~100× rebase

### Pass 2 Refinements

- **Phase Dependency**: Phase 5.4 needs Phase 9's `getSeatCount`—execute Phase 9 first
- **Trial Wiring**: Single chokepoint via `buildActivationParamsFromCheckoutMetadata` + `activateSubscription`
- **isImported Scope**: Stays on Contact, Deal, AND Company (auto-created companies requirement)
- **Seat Backfill**: Required for existing dual-counter rows + Stripe line items
- **Enforcement Flag**: New gates must respect `isSubscriptionEnforcementEnabled()`
- **Settings Route**: `/settings/billing` (tab under account, not standalone `/settings/ai-credits/me`)
- **Deferred Edges**: One-time pack refunds/disputes out of scope for v1

```typescript
private async routeEvent(event: Stripe.Event): Promise<void> {
  const eventMap = {
    'checkout.session.completed': this.handleCheckoutCompleted,
    'customer.subscription.updated': this.handleSubscriptionUpdated,
    'customer.subscription.deleted': this.handleSubscriptionDeleted,
    'invoice.payment_succeeded': this.handleInvoicePaymentSucceeded,
    'invoice.payment_failed': this.handleInvoicePaymentFailed,
    // NEW: One-time credit pack purchases
    'charge.succeeded': this.handleChargeSucceeded,
  };

  const handler = eventMap[event.type];
  if (handler) {
    await handler.call(this, event);
  }
}

private async handleCheckoutCompleted(
  event: Stripe.Event,
): Promise<void> {
  const session = event.data.object as Stripe.Checkout.Session;

  // Existing subscription activation path
  if (session.mode === 'subscription') {
    await this.fulfillCheckoutSession(session);
    return;
  }

  // NEW: One-time credit pack purchase
  if (session.mode === 'payment' && session.metadata?.type === 'credit_pack') {
    await this.fulfillCreditPackPurchase(session);
    return;
  }

  // NEW: Storage pack purchase (recurring, but not subscription-tier)
  if (session.mode === 'subscription' && session.metadata?.type === 'storage_pack') {
    await this.fulfillStoragePackPurchase(session);
    return;
  }
}

private async handleChargeSucceeded(event: Stripe.Event): Promise<void> {
  const charge = event.data.object as Stripe.Charge;

  // Handle credit pack charge (for customers with existing subscriptions)
  if (charge.metadata?.type === 'credit_pack') {
    const customerId = charge.customer as string;
    const organization = await this.findOrgByStripeCustomer(customerId);

    if (organization) {
      const credits = parseInt(charge.metadata.credits, 10);
      await this.creditMeteringService.addOrgCredits(
        organization.id,
        credits,
        'one_time_purchase',
      );
    }
  }
}
```

### 1.3 Credit Pack Fulfillment Service

**File**: `backend/src/subscription/services/credit-pack-fulfillment.service.ts`

```typescript
@Injectable()
export class CreditPackFulfillmentService {
  constructor(
    private readonly creditMeteringService: CreditMeteringService,
    private readonly organizationRepository: EntityRepository<Organization>,
  ) {}

  async fulfillCreditPackPurchase(
    session: Stripe.Checkout.Session,
  ): Promise<void> {
    const orgId = session.metadata?.organizationId;
    const credits = parseInt(session.metadata?.credits || '0', 10);

    if (!orgId || !credits) {
      throw new BadRequestException('Invalid credit pack metadata');
    }

    const organization = await this.organizationRepository.findOne(orgId);
    if (!organization) {
      throw new NotFoundException('Organization not found');
    }

    // Add credits to org pool
    await this.creditMeteringService.addOrgCredits(
      orgId,
      credits,
      'one_time_purchase',
      {
        stripeSessionId: session.id,
        amountPaid: session.amount_total,
        currency: session.currency,
      },
    );

    this.logger.log(
      `Fulfilled credit pack: ${credits} credits for org ${orgId}`,
    );
  }
}
```

```typescript
private buildActivationParamsFromCheckoutMetadata(
  session: Stripe.Checkout.Session,
  stripeSubscription: Stripe.Subscription,
): ActivationParams {
  const tier = session.metadata?.tier as PlanTier;

  // Derive trial state from Stripe subscription
  const status = stripeSubscription.status === 'trialing'
    ? SubscriptionStatus.TRIALING
    : SubscriptionStatus.ACTIVE;

  const trialEndsAt = stripeSubscription.trial_end
    ? new Date(stripeSubscription.trial_end * 1000)
    : null;

  return {
    organizationId: session.metadata.organizationId,
    tier,
    status,
    stripeSubscriptionId: stripeSubscription.id,
    stripeCustomerId: session.customer as string,
    currentPeriodStart: new Date(stripeSubscription.current_period_start * 1000),
    currentPeriodEnd: new Date(stripeSubscription.current_period_end * 1000),
    trialEndsAt,
    seats: this.extractSeatsFromSubscription(stripeSubscription),
  };
}

private extractSeatsFromSubscription(
  subscription: Stripe.Subscription,
): number {
  // For now, single seat SKU—sum all metered/licensed quantities
  return subscription.items.data.reduce(
    (total, item) => total + (item.quantity || 0),
    0,
  );
}
```

### 3.1 Create `CreditMeteringService`

**File**: `backend/src/subscription/services/credit-metering.service.ts`

<Note>
Owns unified wallet, wraps `CreditService`, consolidates `AiUsageAccount` + `AiCreditUsage` into single store.
</Note>

```typescript
export interface CreditBalance {
  orgPool: number;
  personalBalance: number;
  effectiveBalance: number; // orgPool + personalBalance
  orgAllowance: number; // Plan limit × seats
  userCeiling?: number; // Per-user cap (if set)
}

export interface SpendContext {
  userId: string;
  organizationId: string;
  source: 'assistant' | 'agent' | 'messaging';
  channel?: 'org_level' | 'personal';
}

@Injectable()
export class CreditMeteringService {
  constructor(
    private readonly creditService: CreditService,
    private readonly subscriptionService: SubscriptionService,
    private readonly userAllocationRepository: EntityRepository<UserCreditAllocation>,
    private readonly configService: ConfigService,
  ) {}

  async getBalance(context: SpendContext): Promise<CreditBalance> {
    const enforcementEnabled = this.isSubscriptionEnforcementEnabled();

    // When enforcement is OFF, return sentinel values
    if (!enforcementEnabled) {
      return {
        orgPool: Number.MAX_SAFE_INTEGER,
        personalBalance: Number.MAX_SAFE_INTEGER,
        effectiveBalance: Number.MAX_SAFE_INTEGER,
        orgAllowance: Number.MAX_SAFE_INTEGER,
        userCeiling: undefined,
      };
    }

    const [orgBalance, personalBalance, allocation] = await Promise.all([
      this.creditService.getOrgBalance(context.organizationId),
      this.creditService.getUserBalance(context.userId),
      this.getUserAllocation(context.userId, context.organizationId),
    ]);

    const subscription = await this.subscriptionService.findByOrganization(
      context.organizationId,
    );

    const orgAllowance = subscription
      ? subscription.plan.aiCreditsPerMonth * subscription.seats
      : 0;

    return {
      orgPool: orgBalance,
      personalBalance,
      effectiveBalance: orgBalance + personalBalance,
      orgAllowance,
      userCeiling: allocation?.ceiling,
    };
  }

  async deductCredits(
    context: SpendContext,
    amount: number,
  ): Promise<void> {
    if (!this.isSubscriptionEnforcementEnabled()) {
      return; // No-op when enforcement disabled
    }

    const balance = await this.getBalance(context);

    // Org-level channel (auto-reply): org-only, no personal fallback
    if (context.channel === 'org_level') {
      if (balance.orgPool < amount) {
        throw new AiBudgetExceededError('Insufficient org credits');
      }
      await this.creditService.deductOrgCredits(
        context.organizationId,
        amount,
      );
      return;
    }

    // User-driven: check ceiling, then org → personal cascade
    const userSpent = await this.creditService.getUserOrgSpent(
      context.userId,
      context.organizationId,
    );

    const effectiveUserLimit = balance.userCeiling
      ? Math.min(balance.userCeiling - userSpent, balance.orgPool)
      : balance.orgPool;

    if (effectiveUserLimit >= amount) {
      // Spend from org pool
      await this.creditService.deductOrgCredits(
        context.organizationId,
        amount,
      );
      await this.creditService.incrementUserOrgSpent(
        context.userId,
        context.organizationId,
        amount,
      );
    } else {
      // Cascade to personal
      const orgPortion = Math.max(0, effectiveUserLimit);
      const personalPortion = amount - orgPortion;

      if (balance.personalBalance < personalPortion) {
        throw new AiBudgetExceededError('Insufficient credits');
      }

      if (orgPortion > 0) {
        await this.creditService.deductOrgCredits(
          context.organizationId,
          orgPortion,
        );
        await this.creditService.incrementUserOrgSpent(
          context.userId,
          context.organizationId,
          orgPortion,
        );
      }

      await this.creditService.deductUserCredits(
        context.userId,
        personalPortion,
      );
    }
  }

  async addOrgCredits(
    organizationId: string,
    amount: number,
    source: 'subscription_renewal' | 'one_time_purchase',
    metadata?: Record<string, any>,
  ): Promise<void> {
    await this.creditService.addOrgCredits(
      organizationId,
      amount,
      source,
      metadata,
    );
  }

  async setUserCeiling(
    userId: string,
    organizationId: string,
    ceiling: number | null,
  ): Promise<void> {
    // No validation against org allowance (ceilings may oversubscribe)
    await this.userAllocationRepository.upsert({
      userId,
      organizationId,
      ceiling,
    });
  }

  private isSubscriptionEnforcementEnabled(): boolean {
    return this.configService.get<boolean>(
      'SUBSCRIPTION_ENFORCEMENT_ENABLED',
      true,
    );
  }

  private async getUserAllocation(
    userId: string,
    organizationId: string,
  ): Promise<UserCreditAllocation | null> {
    return this.userAllocationRepository.findOne({ userId, organizationId });
  }
}
```

### 4.3 Entity Cap Guard Service

**File**: `backend/src/subscription/services/entity-cap-guard.service.ts`

```typescript
export enum EntityType {
  LEAD = 'lead',
  CONTACT = 'contact',
  COMPANY = 'company',
  DEAL = 'deal',
}

@Injectable()
export class EntityCapGuardService {
  constructor(
    private readonly subscriptionService: SubscriptionService,
    private readonly em: EntityManager,
    private readonly configService: ConfigService,
  ) {}

  async assertCanCreate(
    organizationId: string,
    entityType: EntityType,
    isUserInitiated: boolean,
  ): Promise<void> {
    if (!this.isSubscriptionEnforcementEnabled()) {
      return;
    }

    const subscription = await this.subscriptionService.findByOrganization(
      organizationId,
    );

    const limit = subscription?.plan.inventory?.[`${entityType}s`];
    if (!limit) {
      return; // No cap on this tier
    }

    const currentCount = await this.getEntityCount(
      organizationId,
      entityType,
    );

    if (currentCount >= limit) {
      if (isUserInitiated) {
        throw new ForbiddenException(
          `${entityType} limit reached (${limit} for ${subscription.plan.name} plan)`,
        );
      } else {
        // Soft-degrade: allow create but emit signal
        this.logger.warn(
          `Entity cap exceeded for org ${organizationId} (${entityType}): ${currentCount}/${limit}`,
        );
      }
    }
  }

  private async getEntityCount(
    organizationId: string,
    entityType: EntityType,
  ): Promise<number> {
    const tableName = `${entityType}s`;
    const isImportedField = entityType === 'lead' ? 'lead_source' : 'is_imported';

    const whereClause = entityType === 'lead'
      ? `organization_id = $1 AND is_deleted = FALSE AND lead_source != 'IMPORTED'`
      : `organization_id = $1 AND is_deleted = FALSE AND is_imported = FALSE`;

    const result = await this.em.getConnection().execute(
      `SELECT COUNT(*) FROM ${tableName} WHERE ${whereClause}`,
      [organizationId],
    );

    return parseInt(result[0].count, 10);
  }

  private isSubscriptionEnforcementEnabled(): boolean {
    return this.configService.get<boolean>(
      'SUBSCRIPTION_ENFORCEMENT_ENABLED',
      true,
    );
  }
}
```

```typescript
@Injectable()
export class AiCostCalculatorService {
  calculateCost(usage: {
    model: string;
    inputTokens: number;
    outputTokens: number;
  }): number {
    const costMap = AI_COST_MAP[usage.model];
    if (!costMap) {
      this.logger.warn(`No cost map for model: ${usage.model}, using fallback`);
      return 0; // Or default rate
    }

    const inputCost = (usage.inputTokens / 1000) * costMap.inputPer1k;
    const outputCost = (usage.outputTokens / 1000) * costMap.outputPer1k;

    return Math.ceil(inputCost + outputCost);
  }

  calculateValuationCost(valuationCount: number): number {
    return valuationCount * VALUATION_COST_PER_CALL;
  }
}
```

```typescript
async streamChat(dto: ChatDto, user: User): Promise<ReadableStream> {
  const context = this.buildSpendContext(user, dto);

  // Pre-deduct estimated cost
  const estimatedCost = 500; // Conservative estimate
  await this.creditMeteringService.deductCredits(context, estimatedCost);

  const stream = await this.openAiService.createStreamCompletion({
    messages: dto.messages,
    model: dto.model,
  });

  // Track actual usage
  let actualCost = 0;
  const transformStream = new TransformStream({
    transform: (chunk, controller) => {
      // Accumulate usage from chunks
      if (chunk.usage) {
        actualCost = this.aiCostCalculator.calculateCost(chunk.usage);
      }
      controller.enqueue(chunk);
    },
    flush: async () => {
      // Refund difference
      const refund = estimatedCost - actualCost;
      if (refund > 0) {
        await this.creditMeteringService.addOrgCredits(
          user.organizationId,
          refund,
          'stream_refund',
        );
      }
    },
  });

  return stream.pipeThrough(transformStream);
}
```

```typescript
@Injectable()
export class StorageGuardService {
  constructor(
    private readonly subscriptionService: SubscriptionService,
    private readonly configService: ConfigService,
  ) {}

  async assertCanUpload(
    organizationId: string,
    fileSizeMb: number,
    isUserInitiated: boolean,
  ): Promise<void> {
    if (!this.isSubscriptionEnforcementEnabled()) {
      return;
    }

    const subscription = await this.subscriptionService.findByOrganization(
      organizationId,
    );

    if (!subscription) {
      throw new ForbiddenException('No active subscription');
    }

    const fileSizeGb = fileSizeMb / 1024;
    const projectedUsage = subscription.storageGbUsed + fileSizeGb;

    if (projectedUsage > subscription.storageGbLimit) {
      if (isUserInitiated) {
        throw new ForbiddenException(
          `Storage limit reached (${subscription.storageGbLimit} GB)`,
        );
      } else {
        // Soft-degrade: allow upload but emit signal
        this.logger.warn(
          `Storage cap exceeded for org ${organizationId}: ${projectedUsage}/${subscription.storageGbLimit} GB`,
        );
      }
    }
  }

  async incrementUsage(
    organizationId: string,
    fileSizeMb: number,
  ): Promise<void> {
    const subscription = await this.subscriptionService.findByOrganization(
      organizationId,
    );

    if (subscription) {
      subscription.storageGbUsed += fileSizeMb / 1024;
      await this.subscriptionService.flush();
    }
  }

  private isSubscriptionEnforcementEnabled(): boolean {
    return this.configService.get<boolean>(
      'SUBSCRIPTION_ENFORCEMENT_ENABLED',
      true,
    );
  }
}
```

### 10.4 Credit Top-Up Flow

**Component**: `CreditTopUpDialog.tsx`

```tsx
export function CreditTopUpDialog() {
  const [selectedPack, setSelectedPack] = useState<CreditPack | null>(null);

  const packs: CreditPack[] = [
    { credits: 5000, price: 50, priceId: 'price_xxx' },
    { credits: 25000, price: 200, priceId: 'price_xxx' },
    { credits: 100000, price: 750, priceId: 'price_xxx' },
  ];

  const handlePurchase = async () => {
    const { sessionUrl } = await api.post('/subscriptions/checkout', {
      priceId: selectedPack.priceId,
      mode: 'payment',
      metadata: { type: 'credit_pack', credits: selectedPack.credits },
    });

    window.location.href = sessionUrl;
  };

  return (
    <Dialog>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Buy AI credits</DialogTitle>
        </DialogHeader>

        <div className="space-y-4">
          {packs.map((pack) => (
            <Card
              key={pack.credits}
              className={cn(
                'cursor-pointer',
                selectedPack === pack && 'ring-2 ring-primary',
              )}
              onClick={() => setSelectedPack(pack)}
            >
              <CardContent className="p-4">
                <p className="font-semibold">
                  {pack.credits.toLocaleString()} credits
                </p>
                <p className="text-sm text-muted-foreground">
                  {pack.price} AED
                </p>
              </CardContent>
            </Card>
          ))}
        </div>

        <DialogFooter>
          <Button onClick={handlePurchase} disabled={!selectedPack}>
            Purchase
          </Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```
