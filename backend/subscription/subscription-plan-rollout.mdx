# Subscription Plan Rollout

This document outlines the comprehensive rollout plan for implementing Free/Pro/Business subscription packaging in the PropWise CRM system. The plan is structured in phases, prioritizing backend implementation followed by frontend integration.

<Note>
This rollout implements the subscription packaging strategy with verified code-level analysis and addresses 12 identified gaps through a phased approach.
</Note>

## Overview

The subscription plan rollout introduces a three-tier system (Free/Pro/Business) with unified credit wallets, seat-based billing, and comprehensive usage controls. The implementation follows a backend-first approach to ensure data integrity and system stability.

### Key Components
- **Three-tier subscription model**: Free, Pro, Business
- **Unified credit wallet system**: Consolidates AI and messaging credits
- **Seat-based billing**: Single per-agent SKU model
- **90-day trial period**: For Pro and Business tiers
- **Usage caps and enforcement**: Entity limits and credit controls

export class StorageMeteringService {
  async checkStorageLimit(orgId: string, additionalSize: number): Promise<void> {

  const subscription = await stripe.subscriptions.retrieve(subscriptionId);
  
  // Calculate total seats from existing line items
  const totalSeats = subscription.items.data.reduce((sum, item) => {
    if (item.price.metadata.type === 'seat') {
      return sum + (item.quantity || 0);
    }
    return sum;
