## Overview

The Bayut Pull API enables clients to retrieve comprehensive lead data from the Bayut platform. This API provides access to:

- **Lead Data**: Actual inquiries and contact information from potential customers
- **View Data**: Analytics on how many times listings, agents, or agencies were viewed
- **Multiple Communication Channels**: Support for email, WhatsApp, SMS, and phone interactions
- **Flexible Targeting**: Query leads by listing, agent, or agency
- **Call Logs and Story Leads**: Additional specialized lead types

<Note>
The API uses a bearer token authentication system where each client receives a unique API key for secure access.
</Note>

## Base Endpoint

```
www.bayut.com/api-v7/stats/website-client-leads
```

## Authorization

Each client receives a unique API key that must be included as a bearer token in the `Authorization` header:

```http
Authorization: Bearer <API_KEY>
```

<Warning>
Keep your API key secure and never expose it in client-side code or public repositories.
</Warning>

<Tip>
The `timestamp` parameter should be URL-encoded. For example, `2025-03-10 00:00:00` becomes `2025-03-10%2000%3A00%3A00` where `%20` represents spaces and `%3A` represents colons.
</Tip>

## Query Parameters

| Parameter | Description | Required | Possible Values |
|-----------|-------------|----------|-----------------|
| `type` | The type of lead/interaction | Yes | `whatsapp`, `sms`, `phone`, `email`, `call_logs`, `story_leads` |
| `timestamp` | The date and time threshold for results | Yes | Format: `YYYY-MM-DD HH:MM:SS` |
| `target` | The entity being targeted | Yes* | `listing`, `agent`, `agency` |
| `is_trulead` | Whether to return actual leads or view counts | Yes | `0` (views), `1` (leads) |

<Note>
*The `target` parameter is not required for `call_logs` and `story_leads` types.
</Note>

### Parameter Details

- **`is_trulead=1`**: Returns actual lead data with contact information and messages
- **`is_trulead=0`**: Returns view count statistics without personal information
- **`timestamp`**: Acts as a filter to retrieve data from this date/time onwards

## Error Handling

<AccordionGroup>
<Accordion title="Authentication Errors">

**401 Unauthorized**: Invalid or missing API key
```json
{
  "error": "Unauthorized",
  "message": "Invalid API key"
}
```

**Solution**: Verify your API key is correct and included in the Authorization header.

</Accordion>
<Accordion title="Parameter Errors">

**400 Bad Request**: Invalid query parameters
```json
{
  "error": "Bad Request",
  "message": "Invalid timestamp format"
}
```

**Solution**: Ensure all required parameters are included and properly formatted.

</Accordion>
<Accordion title="Rate Limiting">

**429 Too Many Requests**: API rate limit exceeded
```json
{
  "error": "Rate Limit Exceeded",
  "message": "Too many requests"
}
```

**Solution**: Implement proper rate limiting in your application and retry with exponential backoff.

</Accordion>
</AccordionGroup>

## Integration Examples

<CodeGroup>
```python Python Integration
import requests
from datetime import datetime
import time

class BayutAPI:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = "www.bayut.com/api-v7/stats/website-client-leads"
        self.headers = {"Authorization": f"Bearer {api_key}"}
    
    def get_leads(self, timestamp, target, lead_type, is_trulead=1):
        params = {
            "timestamp": timestamp,
            "target": target,
            "type": lead_type,
            "is_trulead": str(is_trulead)
        }
        
        try:
            response = requests.get(
                self.base_url,
                headers=self.headers,
                params=params
            )
            response.raise_for_status()
            return response.json()
        except requests.exceptions.RequestException as e:
            print(f"API request failed: {e}")
            return None
    
    def get_all_leads(self, timestamp):
        """Get all types of leads for a given timestamp"""
        lead_types = ["email", "whatsapp", "sms", "phone"]
        targets = ["listing", "agent", "agency"]
        all_leads = {}
        
        for lead_type in lead_types:
            for target in targets:
                key = f"{lead_type}_{target}"
                all_leads[key] = self.get_leads(
                    timestamp, target, lead_type, is_trulead=1
                )
                time.sleep(0.1)  # Rate limiting
        
        return all_leads

# Usage
api = BayutAPI("your-api-key-here")
leads = api.get_leads("2025-03-10 00:00:00", "listing", "email")
```

```javascript Node.js Integration
const axios = require('axios');

class BayutAPI {
    constructor(apiKey) {
        this.apiKey = apiKey;
        this.baseURL = 'www.bayut.com/api-v7/stats/website-client-leads';
        this.headers = {
            'Authorization': `Bearer ${apiKey}`
        };
    }

    async getLeads(timestamp, target, type, isTrueLead = 1) {
        const params = {
            timestamp,
            target,
            type,
            is_trulead: isTrueLead.toString()
        };

        try {
            const response = await axios.get(this.baseURL, {
                headers: this.headers,
                params
            });
            return response.data;
        } catch (error) {
            console.error('API request failed:', error.message);
            throw error;
        }
    }

    async getAllLeadTypes(timestamp) {
        const leadTypes = ['email', 'whatsapp', 'sms', 'phone'];
        const targets = ['listing', 'agent', 'agency'];
        const results = {};

        for (const type of leadTypes) {
            for (const target of targets) {
                const key = `${type}_${target}`;
                try {
                    results[key] = await this.getLeads(timestamp, target, type);
                    // Rate limiting
                    await new Promise(resolve => setTimeout(resolve, 100));
                } catch (error) {
                    console.error(`Failed to fetch ${key}:`, error.message);
                    results[key] = null;
                }
            }
        }

        return results;
    }
}

// Usage
const api = new BayutAPI('your-api-key-here');
api.getLeads('2025-03-10 00:00:00', 'listing', 'email')
   .then(leads => console.log(leads))
   .catch(error => console.error(error));
```
